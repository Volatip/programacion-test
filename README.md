# Programación Médica

Plataforma interna para gestionar la programación de funcionarios, la carga de datos operativos y la revisión de métricas asociadas por período.

## Stack tecnológico

- Frontend: React 18, TypeScript, Vite, Tailwind CSS, React Router, Zustand, Recharts
- Backend: FastAPI, SQLAlchemy, Alembic, Pydantic
- Base de datos: PostgreSQL
- Pruebas: Vitest, Testing Library, Pytest
